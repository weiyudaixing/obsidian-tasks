## Supported statuses

<!-- snippet: DocsSamplesForStatuses.test.Theme_Border_Text.approved.txt -->
```txt
- [ ] #task `space` To Do
- [/] #task `/` In Progress
- [x] #task `x` Done
- [-] #task `-` Cancelled
- [>] #task `>` Rescheduled
- [<] #task `<` Scheduled
- [!] #task `!` Important
- [?] #task `?` Question
- [i] #task `i` Infomation
- [S] #task `S` Amount
- [*] #task `*` Star
- [b] #task `b` Bookmark
- [“] #task `“` Quote
- [n] #task `n` Note
- [l] #task `l` Location
- [I] #task `I` Idea
- [p] #task `p` Pro
- [c] #task `c` Con
- [u] #task `u` Up
- [d] #task `d` Down
```
<!-- endSnippet -->
